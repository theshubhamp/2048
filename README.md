# 2048

A clean, responsive 2048 clone built with plain HTML, CSS, and JavaScript — no frameworks, no dependencies.

**Open `index.html` in any browser to play.** Use arrow keys, WASD, or swipe on mobile.

## Features

- Smooth tile appear/pop animations
- Score and persistent high score (localStorage)
- Touch-friendly for mobile
